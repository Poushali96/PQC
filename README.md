# Adaptive Post-Quantum Cryptography (PQC) for 6G V2X Networks  
### Reproducible Implementation for FutureG/NDSS Workshop Submission

This repository contains a fully reproducible Python implementation of our adaptive Post-Quantum Cryptography (PQC) framework for 6G vehicular networks. The system integrates real ERA5 weather data, LuST-style mobility traces, dynamic 3GPP channel simulation, and an evolutionary optimizer (APMOEA) to select PQC algorithms based on context.

The implementation accompanies our workshop paper:

> **Adaptive PQC for 6G Vehicular Networks: Context-Aware Optimization and Monotonic Security Upgrades**  
> (FutureG / NDSS Workshop Submission)
